## Tech Stack
```
Solid, Vite, Tailwind CSS
```

## First

```bash
$ npm install
```

### Learn more on the [Solid Website](https://solidjs.com) 
### `npm run dev`

Runs the application frontend on [http://localhost:5173](http://localhost:5173)

### `npm run build`

Builds the app for production to the `dist` folder.
